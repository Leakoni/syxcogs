# SyxCogs
RED Bot - COGS to make things more fun!
## Installation
```
[p]cog repo add Syxcogs https://github.com/javascripto/syxcogs
[p]cog install Syxcogs [cog]
```
Example:
```
[p]cog install Syxcogs hug
```
## Usage
```
[p]hug
[p]bite
[p]mad
[p]punch
ect
```
